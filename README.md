BudgetFlow is a full-stack personal budgeting application designed to simplify financial tracking and provide users with an organized, user-friendly dashboard experience.
Live Demo

Features
User authentication and account management
Expense and transaction tracking
Category and subcategory customization
Interactive dashboards and analytics
Savings and balance tracking
Responsive UI for desktop and mobile
Currency customization support

Technologies Used
Next.js
React
Supabase
JavaScript / TypeScript
CSS
GitHub

Purpose
This project was created to build a practical budgeting solution for small businesses and individuals while gaining hands-on experience with full-stack web development, authentication systems, database integration, and responsive application design.
